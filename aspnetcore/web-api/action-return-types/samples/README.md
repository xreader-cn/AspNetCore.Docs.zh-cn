| 目录                   | 目标 |

| *2x/WebApiSample.Api.21*    | ASP.NET Core 2.1 Web API 项目。                         |
| *2x/WebApiSample.Api.Pre21* | ASP.NET Core 2.0 Web API 项目。                         |
| *2x/WebApiSample.DataAccess*| .NET Standard 2.0 数据访问层项目，供 ASP.NET Core 2.x Web API 项目使用。|
